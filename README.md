# Factory Freight Connections - Software Proposal Suite

A comprehensive transport management system proposal platform built by Factory Freight Connections.

## Project Overview

This project showcases a next-generation transport management system designed specifically for logistics operators. The platform includes interactive proposal presentations, comprehensive system diagrams, ROI calculations, and deployment timelines.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

### Prerequisites

- Node.js & npm (or Bun)

### Getting Started

```sh
# Clone the repository
git clone https://github.com/oneinawilliam/logistics-proposal-suite.git

# Navigate to the project directory
cd logistics-proposal-suite

# Install dependencies
npm install
# or
bun install

# Start the development server
npm run dev
# or
bun dev
```

The application will be available at `http://localhost:8080`

## Build

```sh
# Production build
npm run build

# Development build
npm run build:dev
```
